  ------------
  HOW TO USE
  ------------

1.  Open index.html in a browser to preview the site locally.
2.  Edit text and content using a code editor such as:
    -   Visual Studio Code (Recommended)
    -   Notepad++
3.  Place all images inside the images/ folder.
4.  Update prices, dog details, and testimonials directly in the HTML
    files.

  ----------------------
  UPLOADING TO HOSTING
  ----------------------

1.  Login to your hosting (cPanel or similar).
2.  Open File Manager.
3.  Navigate to public_html/.
4.  Upload all files and folders INSIDE kabukennels-website.
5.  Ensure index.html is directly inside public_html.

  -------------------------------
  SEO & GOOGLE BUSINESS PROFILE
  -------------------------------

-   Ensure the website URL matches your Google Business Profile.
-   Phone number and email must be the same everywhere.
-   Upload real images to both the website and Google Business Profile.
